About CatDog Breed Classifier

Author: Chris Wang
Contact: s22019.wang@stu.scie.com.cn

CatDog Breed Classifier is an advanced web-based image recognition project using PyTorch for model training, ONNX Runtime for inference, and Flask for user-friendly cat and dog classification, presenting results in Top 5.

    Important Notice:

    - Animal images should be clear and easily recognizable.
    - Percentages after species do not represent real probabilities.
    - Animal images should be clear and easily recognizable.
    - The datasets of model for identifying whether it's a cat or dog does not include other animal species.Some animals with similar appearances to cats or dogs may be misclassified.
    - The model only includes common species of dogs and cats.

Disclaimer:

- Uploaded image files are automatically deleted after classification.
- This project was created as a demonstration of web-based image classification using deep learning models.
- The project is intended for learning purposes and can serve as a starting point for building more complex image classification web applications.
- This website is for reference purposes only.
